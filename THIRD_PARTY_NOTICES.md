## pbakaus/impeccable

- **Repository:** https://github.com/pbakaus/impeccable
- **License:** Apache License 2.0
- **How Opriva uses it:** The seven-domain structure of `skills/opriva-design-fundamentals-auditor/SKILL.md` (typography, color and contrast, spatial design, motion design, interaction design, responsive design, UX writing) is inspired by impeccable's domain decomposition.
- **What was adapted:** The topical decomposition itself — the idea that universal design vocabulary is usefully reviewed as seven separable dimensions.
- **What was NOT adapted:** impeccable's text content, slash commands, `npx impeccable detect` CLI, `.claude/`/`.cursor/`/`.agents/`/`.gemini/` directory distribution, settings registration, or any executable behavior.
- **Status:** Inspirational reference only. The Opriva lens content is original, written in Opriva voice and calibrated to enterprise SaaS, MSP / Integrator and Internal IT workspaces, and Opriva's shell / drawer / table conventions.
